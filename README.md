# nvim-kaysum

Neovim distribution built on top of [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [LazyVim documentation](https://lazyvim.github.io/) to learn more
about the base setup.

## Install

```sh
git clone https://github.com/KaySum/nvim ~/.config/nvim
```

If you want to use this as your own starting point, remove the `.git` folder so
you can add it to your own repo later:

```sh
rm -rf ~/.config/nvim/.git
```
